# 门店调拨扫码工具 - 纯离线版 📱

**单个 HTML 文件，零外部依赖，不联网也能用！**

## 核心特性

| 特性 | 说明 |
|------|------|
| 🔒 **纯本地运行** | 所有数据存手机浏览器 localStorage，不上传任何服务器 |
| 📷 **原生扫码** | 使用浏览器内置 BarcodeDetector API，不加载任何外部 JS 库 |
| 📱 **跨平台** | Android Chrome 和 iOS Safari 16.4+ 原生支持 |
| 🏠 **添加到桌面** | PWA 支持，可以像真APP一样放桌面 |
| 📥 **导出CSV** | 数据可导出到本地，Excel 打开 |

## 怎么用

### 方式一：直接发文件到手机
把这个 `index.html` 发到手机上（微信/QQ/数据线），用浏览器打开即可。

### 方式二：添加到桌面（推荐）

**Android：** Chrome 打开 → 菜单 → 添加到主屏幕

**iOS 26：** Safari 打开 → 共享 → 添加到主屏幕

## ⚠️ 提醒

- 数据存在手机浏览器本地，**清缓存会丢失**
- 建议**定期导出CSV**备份
- 扫码功能需要**授予摄像头权限**
- 支持条码格式：Code 128 / EAN-13 / EAN-8 / UPC / Code 39 / QR 码等
