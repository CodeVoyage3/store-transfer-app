# 门店调拨扫码工具 📱

**纯离线版 — 单个 HTML 文件，零外部依赖，不联网也能用！**

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 📋 **调拨基本信息** | 填写单号、调出/调入门店、入库单号、调拨类型、备注、店长、经手人 |
| 📊 **商品库导入** | 从 Excel 导入商品数据（含条码、名称、商品编码） |
| 📦 **箱管理** | 多箱分箱管理，切换/新建箱子 |
| 📷 **扫码录入** | 摄像头扫码 + 手动输入条码，同单号同箱同条码自动叠加数量 |
| 🧾 **记录管理** | 分组查看、搜索筛选、编辑删除、双重确认清空 |
| 📤 **分享/下载** | 分享为 CSV（钉钉兼容），下载为正式 XLSX（带合并单元格） |
| 💾 **自动保存** | 数据实时存入浏览器 localStorage |
| 🔄 **自动清空** | 超过 15 天未打开页面，自动清空所有数据 |

## 快速上手

### 第一步：填写调拨基本信息

点击 **扫一扫** 图标弹出调拨信息填写弹窗，填写完成后自动进入扫码页。

**必填项：** 调拨单号、调出门店、调入门店、调拨类型、店长、经手人

**示例：**
> 调拨单号：RVKKT20240601-001
> 调出门店：东莞壹号广场KKV店
> 调入门店：东莞厚街万达广场KKV店
> 门店入库单号：TPA4000001298231

锁定态下只显示单号和调出/调入信息，点击 **✏️ 修改** 可查看和编辑全部字段。

### 第二步：导入商品库

在扫码页的 **📊 商品库** 区域点击 **📥 导入**，选择 Excel 文件。

**商品库为空时** 页面会提示 "💡 暂无商品，请先导入 Excel 商品信息"。

**Excel 格式要求：**
- 必须包含 **条形码**（或 条码）列和 **商品名称**（或 名称）列
- 可选列：**商品编码**、**基本单位**（或 单位）
- 支持多个条码用逗号分隔（自动拆分为多条）
- 自动跳过重复条码和以 ZB 结尾的条码

导入成功后自动提示导入数量。也可点击 **🗑 清空** 双重确认后清空商品库。

### 第三步：扫码录入

1. 点击 **扫一扫** 图标启动摄像头
2. 将条码对准扫描框，扫码成功后有震动+提示音
3. 输入**数量**后自动提交（按回车或失焦）

**手动输入条码：** 在下方的输入框输入条码，点击 **查找** 按钮。

**叠加逻辑：** 同一调拨单号 + 同一箱号 + 同一条码 → 自动累加数量，不拆分成多条记录。

> 例：A商品 3 件录入第 1 箱，再扫 A 商品 1 件入第 1 箱 → 自动合计为 4 件。

### 第四步：箱管理

- **➕ 新建箱**：点击新建箱子，自动切换到新箱
- **◀ ▶ 切换**：左右箭头切换已创建的箱子
- **状态显示**：显示当前箱的记录条数和总件数

> 必须先在第 1 箱有记录后，才能新建第 2 箱。

### 第五步：查看记录

切换到 **📋 记录** Tab，可查看所有调拨记录。

**记录分组：** 按调拨单号 → 按箱号 层级折叠展示。

**功能：**
- 🔍 搜索：按商品名称/条码/门店搜索
- 🏪 筛选：按调入门店筛选
- ✏️ 点击记录可编辑（修改条码/数量/箱号）
- 🗑 删除单条记录或整张单据（双重确认）
- ✅ 勾选单据后批量导出

### 第六步：导出数据

在记录页勾选要导出的单据后：

| 按钮 | 格式 | 用途 |
|------|------|------|
| 📤 **分享** | CSV | 分享到微信/钉钉等，直接查看 |
| 📥 **下载** | XLSX | 正式 Excel 文件，带合并单元格 |

**导出列说明：** 门店调拨单号 | 货源门店 | 目标门店 | 创建时间 | 调拨类型 | 门店入库单号 | 备注 | **店长** | **经手人** | **商品编码** | 商品条形码 | 商品名称 | 实调数量 | 总计 | 仓库 | 箱号

同单据的非明细字段（单号、门店、时间、类型、入库单号、备注、店长、经手人、总计）自动合并居中。

## 数据安全

### 存储方式
- **全部数据存在手机浏览器本地（localStorage）**
- 不上传任何服务器，不联网也能正常使用

### 自动清空规则
- 页面记录**上次访问时间**
- 超过 **15 天** 未打开页面 → 自动静默清空商品库和调拨记录
- 这是为了保障数据时效性，避免过期数据混淆

### 手动备份/恢复
- 右上角 **💾 自动保存** 指示正常（绿色）/ 异常（黄色）
- 异常时需手动导出备份（代码内置导出/导入 JSON 功能）

### ⚠️ 注意事项
- **清浏览器缓存会丢失所有数据**
- 建议每周导出一次 XLSX 作为备份
- 扫码需**授予摄像头权限**
- 支持条码：EAN-13 / EAN-8 / Code 128（默认只开启商品条码相关码制）

## 安装到桌面（推荐）

**Android（Chrome）：** 打开页面 → 菜单 ⋮ → 添加到主屏幕

**iOS（Safari）：** 打开页面 → 分享 → 添加到主屏幕

添加到桌面后可像原生应用一样全屏运行，体验更好。

## 技术说明

- 纯 HTML + CSS + JavaScript 单文件应用
- 扫码引擎：QuaggaJS（本地库，无需联网，兼容所有现代浏览器）
- XLSX 导出：SheetJS（xlsx.core.min.js）
- 无任何后端服务依赖
- PWA 支持（manifest.json + 图标）
