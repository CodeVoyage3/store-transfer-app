远程商品库同步

📲
门店端（自动同步）
  • ●App 启动时自动从云端拉取加密商品数据
  • ●扫码时自动匹配商品名称，无需手动导入
  • ●数据在内存中解密，门店无法直接查看全量商品
  • ●首次联网同步后，离线也可扫码（本地缓存）
  • ●商品库区域显示"📦 N 条 已同步"即表示成功
🖥️
管理员端（发布数据）
  • ●打开 管理工具.html（电脑浏览器）
  • ●输入管理密码进入 → 配置 GitHub Token
  • ●上传商品 Excel → 自动脱敏（仅留条码+商品名）
  • ●确认数据预览 → 点击"加密并发布"
  • ●数据 AES-256 加密后自动推送到云端
  • ●门店下次打开 App 自动获取最新数据

数据安全须知

💾
纯本地存储
全部数据存手机浏览器，不上传任何服务器
⏰
15天自动清空
超过15天未打开 → 静默清空商品库和记录
🔒
AES-256 加密传输
商品数据云端存储采用 AES-256 加密，门店无法直接查看全量数据
